# tmux

* Reattach an existing session
  ```shell
tmux attach 
  ```

* Detach from tmux (everything running will be in background)
  ```
ctrl+B, D
  ```

* New tmux session
  ```
ctrl+B, C 
  ```

* Switch session
  ```
ctrl+B, 0~9
  ```

* Select a session from the list (Use left and right allow keys to see the list)
  ```shell
ctrl+B, s
  ```

* Terminate the session
```
ctrl+D
```

* scroll
```
ctrl + b + [
```
